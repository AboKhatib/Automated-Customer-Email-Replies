AI Email Support Auto-Responder

This workflow connects Gmail with OpenAI to handle customer emails. It detects unread messages and uses an AI model to draft a response. The system replies directly to the original email thread.

Features
Automated triggers check for new messages
Language processing understands intent
Thread continuity keeps conversations organized

Setup Instructions
Import the JSON file into n8n
Connect Gmail credentials
Add your OpenAI API key
Adjust the prompt to match your rules
